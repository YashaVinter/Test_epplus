### Зависимости  

как то установить себе библиотеку EPPlus - через powershell или nuget  

### Преднастройки программы  

в Main меняется path на свой путь до файла  
sheet_trans и sheet_terminal меняются на свои названия листов  
все остальные значения ячеек и сдвиги также смотрятся по ситуации и/или своему файлу
